Who is it for?

This course is for developers and testers who are brand new to Cucumber. You'll accomplish hands-on technical tasks, like writing code in text editors and running Cucumber from the command-line interface.

You should be reasonably comfortable with these tools or keen to learn about them.

By the end of the course you'll know how to build your own Cucumber suites and how to drive a web browser using Cucumber.

You'll learn how to:

  • Describe requirements using Gherkin, the specification language for Cucumber
  • Drive a browser using Selenium WebDriver
  • Build a framework to structure your Cucumber suite
  • Use tags to organise your tests
  • Use hooks to automate your test setup

Plus many more pro tips directly from the Cucumber core team…

How does it work?

We’ve taught hundreds of people how to use BDD and Cucumber in our in-house courses. We know that practice and reflection are key to learning, so each lesson works like this:

Watch the video

Each lesson introduces a new technique or concept with a short video.

 

Practice the techniques

You’ll be given exercises to try out.

 

Reflect

We’ll ask you a series of thought-provoking questions that will deepen your understanding of the topic.
